# FIRST DAWN QUESTIONS

## Foundational Questions

* What is the smallest honest unit of simulated experience?
* How should time move when no observer is watching?
* Which natural laws must be established before people exist?
* How should memories fade without becoming arbitrary?
* What information can observers see without becoming participants?
* What makes an event historically significant enough to preserve forever?
* How will staging and sandbox worlds prove changes before production continuity is touched?

## Simulation Boundaries

* Which systems belong in the world engine before the citizen engine starts?
* How should weather, geography, resources, and seasons influence people without scripting outcomes?
* What data must be immutable once production history begins?
* What audit trail is required for any administrative action?
